# Changelog

## [Unreleased]

- Prolog parser built on [Parslet](https://kschiess.github.io/parslet/)
- Transformer converting parse trees to domain objects (Atom, Variable, Compound, Rule)
- SLD resolution with recursive backtracking
- Unification of atoms, variables, anonymous variables, and compound terms
- Environment-based variable binding with cycle detection
- Knowledge base and query parsing as separate entry points
- Web example using Roda
